Launchport

Dock magnético para iPad

Launchport es un revolucionario dock para iPad.

El sistema compuesto por una base de mesa o pared, recarga el iPad utilizando un "Shuttle" o forro que permite recargar el iPad usando técnica de inducción mediante magnetos.

Shuttle

Forro con magnetos, sirve para proteger el iPad y fijarlo mediante magnetismo a la base de mesa o pared, disponible en color negro o blanco.

Mejor sonido

El shuttle cuenta con un sistema que enfoca el altoparlante, mejorando el sonido del iPad.

Todo a la mano

El shuttle cuenta con los orificios para manipular los botones del iPad.

Base de mesa

En acabado metálico, recarga el iPad sin cables.

Base de pared

Discreta, permite recargar el iPad sin usar cables y girarlo en posición horizontal o vertical.
