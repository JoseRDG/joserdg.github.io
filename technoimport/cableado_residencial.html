¿Cableado o inalámbrico?

Esta es la confusión mas grande de los tiempos modernos.

En la radio escuchamos Internet inalámbrico a altas velocidades y nos ofrecen sistemas de wifi que ya no necesitan cables.

En realidad los dos son necesarios y se complementan.

La comisión de regulación de comunicaciones expidió la resolución de REDES INTERNAS DE COMUNICACIONES RITEL la cual exige que todas las nuevas construcciones del país cuentan con infraestructura de comunicaciones a partir de Agosto de 2014.

Redes de datos para hogar

En el diagrama de la izquierda se aprecian los componentes de las redes de datos para hogar.

Los operadores de servicios publicos como ETB y Claro instalan un equipo que se conoce como MODEM o CPE.

Dependiendo el MODEM puede tener puertos de conexion cableada por ejemplo para computadores o equipos que solo funcionan conectandose mediante un cable a la red de datos.

Algunos también crean una red inalámbrica conocida como red WIFI para que se puedan conectar equipos inalámbricos.

Si el MODEM no trae sino un puerto para conexión cableada, entonces se requiere instalar un Enrutador. Este equipo cuenta con puertos adicionales para conectar mas equipos de forma cableada.

En espacios de mayor tamaño como casas de 2 pisos y apartamentos de mas de 2 habitaciones, es necesario instalar un punto de acceso inalámbrico que extiende la señal inalámbrica WiFi.

¿Para qué sirven las redes de datos en el hogar?

La mayoría de equipos recientemente lanzados se pueden conectar a las redes de datos para controlarlos desde el teléfonos inteligentes y tabletas.

Muchos de estos equipos se actualizan periódicamente descargando de forma automática las ultimas actualizaciones de Internet.

Las conexiones de datos de alta velocidad le permiten además disfrutar de música y videos de Internet y controlar su hogar remotamente.
